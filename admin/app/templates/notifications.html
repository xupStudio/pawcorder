{{ t('NOTIF_TG_STEP1') }}
    {% if lang == 'zh-TW' %}
  1. 用 Telegram 找 @BotFather。
  2. 傳送 /newbot，自取一個名字、和以 _bot 結尾的 username。
  3. BotFather 會回給你一段 token（像 12345:ABC-defGhij...），複製起來就是 Bot token。
  4. 打開你新建的 bot 對話視窗，按 Start。（bot 必須先有人開始對話才能傳訊。）
    5. {% else %}
  5. In Telegram, message @BotFather.
  6. Send /newbot. Pick a name and a username ending in _bot.
  7. BotFather replies with a token like 12345:ABC-defGhij.... Copy it.
  8. Open a chat with your new bot and tap Start.
    9. {% endif %}
{{ t('NOTIF_TG_STEP2') }}
    {% if lang == 'zh-TW' %}
  1. 在 Telegram 找 @userinfobot。
  2. 它會回給你一個數字 ID（例如 123456789），那就是 chat ID。
  3. 想推到群組的話，把你的 bot 跟 @userinfobot 都加進群組，然後在群組裡發 /start@userinfobot，回傳的負數就是群組 chat ID。
    4. {% else %}
  4. In Telegram, message @userinfobot.
  5. It replies with your numeric ID (e.g. 123456789). That's the Chat ID.
  6. For a group: add your bot AND @userinfobot to the group, then send /start@userinfobot. The negative number is the group chat ID.
    7. {% endif %}

{{ t('NOTIF_LINE_STEP1') }}
    {% if lang == 'zh-TW' %}
  1. 到 LINE Developers Console 用 LINE 帳號登入。
  2. 建立一個 Provider（如果還沒有），底下建立一個「Messaging API」channel。
  3. 建好後到 Messaging API 分頁底部，按「Issue」產生 Channel access token (long-lived)，複製下來。
  4. 同一頁開 Allow bot to join group chats（如果想推群組）。
    5. {% else %}
  5. Sign into LINE Developers Console.
  6. Create a Provider (if you don't have one), then create a "Messaging API" channel under it.
  7. Open the channel's Messaging API tab. Scroll to the bottom and Issue a long-lived Channel access token. Copy it.
  8. On the same tab, allow bots to join groups if you want group notifications.
    9. {% endif %}
{{ t('NOTIF_LINE_STEP2') }}
    {% if lang == 'zh-TW' %}
  1. 用 LINE App 掃描 channel 頁面上方的 QR Code 加你新 bot 為好友。
  2. 對 bot 傳一句話（例如 "hi"）。
  3. 到 channel 的 Messaging API 分頁、最底下的 webhook 區會列出 Recent events 或用 LINE Developers 的 webhook log 看 userId 欄位。複製這串 ID（以 U 開頭）就是 Target ID。
  4. 群組或多人聊天室也可以做為目標，但要先把 bot 邀進去並讀取 webhook log 取得 groupId 或 roomId。
    5. {% else %}
  5. Use LINE on your phone to scan the QR code at the top of the channel page and add your new bot as a friend.
  6. Send your bot any message (e.g. "hi") so it has a destination to push to.
  7. In the channel's Messaging API tab, look at the webhook log to find the userId (starts with U). Copy that — it's your Target ID.
  8. For a group or multi-person chat, add the bot and read the webhook log for groupId or roomId.
    9. {% endif %}
