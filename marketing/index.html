pawcorder

自架寵物攝影機 NVR。影片留你家，雲端用你自己的 Google Drive / Dropbox / NAS。 買硬體一次付清，月費 NT$0。

看 GitHub → 功能總覽 運作原理

一行安裝（Linux · macOS · Windows + WSL2）

$ curl -fsSL https://raw.githubusercontent.com/xupStudio/pawcorder/main/scripts/bootstrap.sh | bash

Linux 自動裝 Docker、Mac 自動裝 Docker Desktop、Windows 走 WSL2 內 Docker Engine。 然後抓 Frigate image、生隨機密碼、選對你硬體的 AI 加速器。 不放心 curl|bash？ 先讀 source。

影片留你家

100% 在家裡網路內處理。雲端備份用你自己的 Google Drive / Dropbox / NAS。pawcorder 看不到、也存不到你的影像。

買硬體 → 0 月費

Furbo 4 支 3 年 NT$48,000。pawcorder 一次性 NT$10,000–19,000，之後 NT$0。差價買台 iPad。

寵物個體辨識

不只「有狗出現」，是「Mochi 今天出現 47 次」。MobileNet ONNX 本機推論，跨攝影機追同一隻。

功能總覽

所有功能都已實作、有單元測試、MIT 開源。

📹 錄影與 AI

攝影機相容

10+ 主流品牌一鍵自動設定（見下方相容表），其他任何 ONVIF / RTSP IP cam 也能接。

硬體加速自動選擇

Intel iGPU 用 OpenVINO、NVIDIA 用 TensorRT、Coral 用 Edge TPU、Pi5 + Hailo 用 Hailo、其他走 CPU。

即時直播

後台儀表板與單機畫面內建 HLS / WebRTC 播放器，支援雙向對講（按住按鈕）。

音訊事件

Frigate 內建音訊模型偵測吠叫、喵叫、玻璃破裂、尖叫、煙霧警報，跟視覺事件走同一條推播。

🐾 寵物個體辨識

幾張照片教會它

每隻寵物上傳 3–5 張照片，MobileNetV3 抽 576 維向量，餘弦相似度 ≥ 0.78 才認定。

跨攝影機時間軸

每隻寵物的一日活動軌跡，跨所有攝影機合併在一條時間軸上。

回溯辨識

新增寵物後一鍵套用到過去 7 天的事件，補回原本叫「unknown」的紀錄。

健康異常警示

每隻寵物有自己的活動量基準，z-score ≥ 2σ 偏低或低於硬性下限就推播提醒。

🖥️ 後台介面

多使用者 + 角色權限

admin / family / kid 三層。最後一個 admin 不能刪不能降，避免把自己鎖在外面。

儀表板

即時縮圖、今日精選時刻、最近事件、Frigate log 即時瀏覽、CPU/RAM/磁碟狀態。

區域編輯器

Canvas 拖點畫多邊形，分為偵測區域（橘）與隱私遮罩（紅）。直接寫進 Frigate 設定。

PTZ 預設位置

把鏡頭轉到「餵食點」「貓砂盆」按下儲存，下次一鍵跳回。走 ONVIF 標準。

隱私模式

手機在家裡 Wi-Fi 自動暫停錄影，回到外網才恢復。也支援手動或排程。

每日精選 + 縮時影片

凌晨 2 點自動 ffmpeg stream-copy 出 30 秒精華（保留 14 天）+ 每分鐘抽幀做縮時（保留 30 天）。

活動熱力圖

每支攝影機 30 天的移動密度，疊加在縮圖上的半透明 PNG。

省電模式

設定一個低 FPS / 動態觸發的時段（例：02–06 點），睡覺時 iGPU 不全速跑。

深色模式 + 雙語

繁中 / 英文完整翻譯，日文 / 韓文骨架已開好等翻譯。深色 / 淺色 / 自動三檔。

API key + Web Push

Bearer token 給腳本用，PWA 走 VAPID 推播；iOS 退回 Telegram fallback。

💾 儲存

NAS 掛載精靈

瀏覽器設定 NFS / SMB：先試掛、再寫進 fstab、立即掛上。SMB 密碼放 0600 憑證檔，不寫進 fstab。

雲端備份（rclone）

8 種後端：Drive / Dropbox / OneDrive / B2 / S3 / Wasabi / R2 / WebDAV。可選 AES-256-GCM 加密。

排程備份

每天自動把 cameras.yml / pets.yml / users.yml 等設定上傳到你的雲。

Schema 版本遷移

每個設定檔有 schema_version，啟動時自動跑遷移；升級不會把舊資料弄壞。

🔔 通知與遠端

Telegram + LINE

貓狗出現秒殺通知。Telegram 附快照、LINE 純文字。免 MQTT broker。

PWA 推播

桌面 / Android 走 VAPID 標準推播；iOS 受 Safari 限制自動退回 Telegram。

Frigate Webhook

事件秒級到達，取代輪詢；含 dedup 不會重複推。

Tailscale 一鍵

內建腳本接你的 tailnet，外網看貓不用 DDNS、不用打洞。

🛠️ 維運品質

一行安裝（三平台）

Linux / macOS / Windows + WSL2 都可以 curl | bash 一鍵裝好——含 Docker 自動安裝、密碼產生、detector 選擇、stack 啟動。Mac 上想先試試也行。也提供 USB 預燒映像。

OTA 更新

後台會抓 GitHub Releases 顯示「有新版」徽章。make update 一行更新。

備份 / 還原

系統頁一鍵下載整個後台狀態的加密 tarball，新機器還原直接接手。

乾淨解除安裝

soft reset（保留錄影、清設定）或完全移除，每個檔案清單都列出來給你看。

API 文件頁

後台 /docs/api 有完整端點清單，附 curl / Home Assistant / iOS 捷徑範例。

~525 個自動化測試

FastAPI route smoke、設定 round-trip、i18n key 覆蓋、CSRF / RBAC、NAS fstab idempotency 全有測試。

攝影機相容清單

10 個主流品牌已測過。沒列的品牌只要支援 ONVIF / RTSP 也都能接。

品牌 設定難度 雙向對講 說明
Reolink 完全自動 ✓ E1 / E1 Pro / E1 Outdoor PoE 推薦款。設成 H.264 後直接接好。
Hikvision 完全自動 — ISAPI 自動偵測，含 Annke / LTS / Hilook OEM。
Dahua / Amcrest 完全自動 Amcrest ✓ CGI 自動設定，Amcrest 是 Dahua OEM 共用。
Axis 完全自動 — VAPIX，RTSP 預設開。LW/WV 機型自動辨識為 WiFi。
Foscam 完全自動 — CGI 自動偵測。
Ubiquiti UniFi Protect 需在 UI 開啟 RTSP — Controller 介面內逐 cam 啟用 RTSP；controller-driven 多 cam onboarding 後續版本支援。
TP-Link Tapo 需在 App 開帳號 ✓ 後台會直接顯示 step-by-step 在 Tapo App 內建立 RTSP 帳號的步驟。
Imou 需在 App 開 RTSP — Imou Life App → 本地設定 → RTSP，後台有圖文步驟。
Wyze 需特殊韌體 — 原廠韌體不支援 RTSP，要刷非官方韌體或跑 docker-wyze-bridge。
其他 ONVIF / RTSP ONVIF 自動偵測 + 手動 fallback — 先試 ONVIF Profile S 自動找串流；找不到再手動填 URL。

完全自動 = 後台輸入 IP/帳密後一鍵完成；其他要先在攝影機自家 App 點幾下，後台會給逐步說明。

怎麼運作的

  1. 1

    準備一台小主機 + Reolink E1 Outdoor PoE 攝影機 + PoE 交換器

    主機可以是新買的 N100 迷你 PC（推薦，CP 值高），也可以是你現有的 Pi 5、Synology x86、舊筆電。總價 NT$3,500–19,000 看怎麼配。所有零件 PChome / 蝦皮都有。

  2. 2

    USB 安裝映像 dd 進去開機

    10 分鐘後主機自己跑完 Ubuntu 安裝、Docker 啟動、pawcorder 上線。家裡 Wi-Fi 任一裝置開瀏覽器到 pawcorder.local:8080。

  3. 3

    圖形化精靈 5 步驟

    掃網段找攝影機 → 設儲存 → 設靈敏度 → 改密碼 → 完成。中文介面，每個欄位都有解釋。

  4. 4

    手機 QR Code 一掃連線

    設定頁有 QR Code，掃了就在手機開 Frigate 即時影像。要在外面看？一行指令裝 Tailscale。
