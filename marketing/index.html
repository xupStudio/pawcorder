章節 02 · 功能總覽

這些都已經做完了。

原始碼開源（MIT 授權），任何人都能查證。下面分成 6 個區塊。

i.

錄影與 AI

攝影機相容

10+ 主流品牌一鍵自動設定（見下方相容表），其他主流 IP 攝影機只要有標準串流也都能接。

自動挑最快的 AI 引擎

Intel 內顯、NVIDIA 顯示卡、Coral USB 加速棒、Raspberry Pi 5 + Hailo AI Kit 都自動偵測；沒有加速晶片就用 CPU。安裝時不用選。

即時直播

管理後台跟手機畫面都能即時看，支援雙向對講（按住按鈕說話）。

聲音事件偵測

內建 AI 偵測吠叫、喵叫、玻璃破裂、尖叫、煙霧警報，跟畫面事件用同一條推播管道。

ii.

寵物個體辨識

幾張照片教會它

每隻寵物上傳 3–5 張照片，AI 在你機器上比對特徵，相似度夠高才認定是哪一隻。

跨攝影機時間軸

每隻寵物一整天的活動軌跡，跨所有攝影機合併成一條時間軸。

回溯辨識

新增寵物後一鍵套用到過去 7 天的事件，把原本沒有名字的紀錄補上。

健康異常警示

系統會學每隻寵物平常的活動量；當天明顯偏低（或太久沒看到）就推播提醒。

健康總覽頁

一頁看完每隻寵物的 30 天活動圖、活動時段熱力圖、健康分數。系統有狀況一眼就看到。

看診摘要一鍵列印

去獸醫前按一個按鈕，產生一張 30 天活動 + 圖表 + 異常事件的 A4 摘要，直接列印或存成 PDF 帶去看診。也能傳一個 24 小時失效連結給獸醫自己開。

每週 Telegram 健康摘要

每個禮拜天早上自動推一份「這週每隻寵物狀況」摘要，把零散告警整合成一次閱讀。

iii.

後台介面

多人共用、各自權限

三種角色：管理員、家人、小孩。最後一個管理員無法被刪除或降權，避免不小心把自己鎖在外面。

儀表板

即時縮圖、今日精選時刻、最近事件、系統日誌、CPU / 記憶體 / 硬碟使用狀況一頁看完。

區域編輯器

在畫面上點選圍出範圍，分成「要偵測的區域」（橘色）和「隱私遮罩」（紅色），按存就立即套用。

鏡頭預設位置

把支援轉動的鏡頭對準「餵食點」「貓砂盆」按存，之後一鍵跳回。

隱私模式

偵測到你手機回到家自動暫停錄影，離家後再自動恢復。也可以手動或排程。

每日精選 + 縮時影片

每天凌晨 2 點自動剪一支 30 秒的當日精華（保留 14 天），加一支 24 小時壓縮成 50 秒的縮時（保留 30 天）。

活動熱力圖

每支攝影機 30 天的活動密度，疊在畫面上一目了然哪些位置最常出現。

省電模式

指定時段（例如凌晨 2–6 點）降低錄影品質或只在偵測到動靜時錄，省電又省硬碟。

深色模式 + 雙語

繁體中文 / 英文完整翻譯，日文 / 韓文已開好框架等翻譯。深色 / 淺色 / 跟隨系統三種主題。

金鑰 + 瀏覽器推播

想自己寫腳本接 Pawcorder？建立一把金鑰就能用。瀏覽器推播 Chrome / Firefox 都收得到，iPhone 自動改用 Telegram 確保不漏訊息。

iv.

儲存

NAS 掛載精靈

在瀏覽器填 NAS 資訊，按「測試」確認連得上，再按「設定」開機自動掛載。NAS 密碼用獨立檔保護，權限只有系統管理員看得到。

雲端備份

支援 8 種雲端：Google Drive / Dropbox / OneDrive / Backblaze B2 / Amazon S3 / Wasabi / Cloudflare R2 / WebDAV。要加密也行。

每日自動備份

每天自動把攝影機清單、寵物資料、使用者帳號等設定備份到你的雲端。

設定檔自動升級

每個設定檔有版本號，Pawcorder 升級時會自動轉換舊設定，不用擔心更新壞掉。

v.

通知與遠端

Telegram + LINE

貓狗一出現馬上推播。Telegram 附上快照、LINE 純文字（LINE 不支援圖片）。不需要再額外裝任何中介服務。

瀏覽器推播

桌面 / Android 收得到瀏覽器原生通知；iPhone 因為 Safari 限制較多，自動改用 Telegram 推播。

事件秒級到達

寵物出現後幾秒內就推播，而且不會重複推同一件事。

外網看貓一鍵搞定

用 Tailscale 一鍵建立你手機與家裡的私人連線，不用設定路由器、也不用申請網址。

章節 03 · 攝影機相容

10 個主流品牌已測過。

沒列在表上的品牌，只要支援標準串流協定（ONVIF / RTSP，多數 IP 攝影機都有）都能接。

品牌 設定難度 雙向對講 說明
Reolink 完全自動 ✓ E1 / E1 Pro / E1 Outdoor PoE 推薦款。設成 H.264 後直接接好。
Hikvision 完全自動 — 自動偵測，含 Annke / LTS / Hilook 等同型號代工品牌。
Dahua / Amcrest 完全自動 Amcrest ✓ 自動設定，Amcrest 是 Dahua 同型號代工。
Axis 完全自動 — 標準串流預設開啟，Wi-Fi 機型也能自動辨識。
Foscam 完全自動 — 自動偵測。
Ubiquiti UniFi Protect 需在控制器開串流 — 在 UniFi 控制器介面為每支攝影機開啟串流；未來版本會做成一鍵新增。
TP-Link Tapo 需在 App 建帳號 ✓ 後台會直接顯示步驟教你在 Tapo App 內建立攝影機帳號。
Imou 需在 App 開啟串流 — Imou Life App → 本地設定 → 開啟串流，後台有圖文步驟。
Wyze 需刷自製韌體 — 原廠韌體不支援串流，要刷社群維護的非官方韌體；或在網路上跑社群轉接服務。
其他 IP 攝影機 自動偵測 + 手動填入 — 支援標準串流協定的攝影機都能用：先試自動偵測，找不到再手動填串流網址。

完全自動 = 後台輸入 IP/帳密後一鍵完成；其他要先在攝影機自家 App 點幾下，後台會給逐步說明。

章節 05 · 怎麼運作的

四步上線。

  1. 1

    準備一台小主機 + Reolink E1 Outdoor PoE 攝影機 + PoE 交換器

    主機可以是新買的 Intel N100 迷你 PC（推薦，CP 值高），也可以是你現有的 Raspberry Pi 5、Synology NAS、舊筆電。總價 NT$3,500–19,000 看怎麼配。所有零件 PChome / 蝦皮都買得到。

  2. 2

    USB 隨身碟燒進去開機

    把現成的安裝映像燒到 USB 隨身碟，插到主機開機。10 分鐘後系統自動裝完、Pawcorder 上線。家裡 Wi-Fi 任一裝置開瀏覽器連到 pawcorder.local:8080 就能用。

  3. 3

    圖形化精靈 5 步驟

    掃描家中網路找攝影機 → 選儲存位置 → 設靈敏度 → 改密碼 → 完成。中文介面，每個欄位都有解釋。

  4. 4

    手機掃 QR Code 連線

    設定頁有 QR Code，手機一掃就能開即時影像。在外面想看？一行指令裝 Tailscale 就好（一個免費的私人連線小工具）。
